Datable

A plugin to properly format dates in inputs.

Demo

Month & Year

<input type="text" class="form-control" data-datable="mmyyyy" />

<input type="text" class="form-control" data-datable="yymm" />

Full Date

<input type="text" class="form-control" data-datable="mmddyy" />

<input type="text" class="form-control" data-datable="ddmmyyyy" />

Alternate Seperators

<input type="text" class="form-control" data-datable="mmddyyyy" data-datable-divider=" - " />

<input type="text" class="form-control" data-datable="ddmmyyyy" data-datable-divider="-" />

Install

After downloading datable, just include the file right above the closing <body> tag.

<script type="text/javascript" src="datable.min.js"></script>

That's all!
